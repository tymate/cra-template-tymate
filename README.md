# Tymate - SPA starter pack

Single page application starter pack featuring Margaret as well as our commonly used libraries.

* react-router-dom
* lodash
* date-fns
* react-modal
* react-use
* react-icons
* axios
* humps
* sanitize.css
* styled-components
